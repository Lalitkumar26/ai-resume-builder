# 🚀 AI Resume Builder

**Build Your Perfect Resume in Minutes**  
AI-Powered · ATS-Friendly · Free to Use  

🌐 **Live Demo:** https://autoresumes.netlify.app/

---

## 📌 Overview

AI Resume Builder is a web application that helps users create professional, ATS-friendly resumes quickly and effortlessly. It uses AI to generate high-quality content, saving time and improving resume effectiveness.

---

## ❗ Problem Statement

Creating a strong resume is difficult because:

- ⏳ **Time-Consuming Process**  
  Writing and formatting resumes from scratch takes hours.

- 🤖 **ATS Rejection**  
  Many resumes are rejected by Applicant Tracking Systems before reaching recruiters.

- 🧭 **Lack of Guidance**  
  Freshers struggle to write impactful and professional content.

- 💰 **Expensive Services**  
  Professional resume tools cost $100–$500.

---

## ✨ Features

- 🧠 **AI Content Generation**  
  Generates summaries, experience descriptions, and skills automatically.

- 🎨 **Multiple Templates**  
  - Classic  
  - Modern Sidebar  
  - Creative Bold  

- 📄 **Instant PDF Download**  
  One-click export to high-quality PDF.

- ✅ **ATS-Friendly Design**  
  Templates optimized for hiring systems.

- ⚡ **Live Preview**  
  Real-time editing experience.

- 🔒 **Secure & Free**  
  Firebase-based authentication and database.

---

## 🛠️ Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript  

### AI
- Google Gemini API  

### Backend & Database
- Firebase Auth  
- Firestore DB  

### PDF Export
- html2pdf.js  

---

## 🏗️ Architecture
User Input
↓
Frontend (HTML/CSS/JS)
↓
Firebase (Auth + Firestore)
↓
Google Gemini API (AI Content)
↓
PDF Generation (html2pdf.js)


---

## ⚔️ Challenges & Solutions

| Challenge | Solution |
|----------|----------|
| Inconsistent AI responses | Fixed prompt templates |
| PDF rendering issues | Used html2pdf.js |
| Auto-save breaking | Added debounce (1.5s delay) |
| ATS + design balance | Semantic HTML + layered CSS |
